2. Double-click the file to open the setup window.

4. Click the Finish button when the process ends.

You start by describing the task in plain language. The software creates a map of the process. You can see each step on your screen. If one step involves searching data, the software handles the connection. If another step involves writing code, it generates the text for you.

### Task Planning
The system keeps a list of tasks. It decides the order of operations based on your goal. It creates tiny units of work to avoid errors. You track this progress through the live status view.

### Visual Updates
You see the progress in real time. The software shows activity updates as they happen. This helps you notice if the AI needs more information from you.

The software uses specialized agents to manage specific parts of your project. One agent might focus on file storage, while another agent handles database updates. These agents talk to each other to solve your problems.

### Code Generation
You enter your requirements, and the software writes the necessary instructions. It ensures the code follows standard practices. You can review the output at any time.

## 🛡️ Troubleshooting Common Issues

If the software fails to start or shows an error, check these items:

- Restart the application: Close the window and launch it again from your desktop.
- Check Internet Access: The software needs the internet to fetch instructions from the AI models. Ensure your firewall allows the program to connect.
- Disk Space: Verify that your computer has enough free space to save your projects.
- Database Connection: The app runs a small data service in the background. If you see a database error, wait ten seconds and refresh the screen.
